Testnet Architecture

The ilpv4.dev domain operates a variety of testnet components, some of which are connected to other external servers in order to create an Inteledger Testnet. The system has the following components:

Emmett Connector

Emmett is a Tier1 Connector implemented in Rust via the interledger-rs project. As can be seen in the diagram above, emmett is peered with marty using Ilp-over-Http.

Marty Connector

Marty is a Tier1 Connector implemented in Java via the java-ilp-connector project. As can be seen in the diagram above, marty is peered with emmett and clara using Ilp-over-Http.

Clara Connector

Clara is a Tier1 Connector implemented in Rust via the interledger-rs project. As can be seen in the diagram above, clara is peered with marty using Ilp-over-Http.
